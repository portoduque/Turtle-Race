# Turtle Race

## About
Turtle Race is a fun game where you can bet on the winner turtle. 
Ready, Set, Go!

## Built With

* [Pyhton](https://www.python.org/downloads/)
***
## Usage example
### You can choose the following turtle colors: red, orange, yellow, green, blue or purple.
